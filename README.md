## TL;DR

Open PowerShell with admin privileges (`Win`+`X A`) and run:

```powershell
irm pauw.io/bro | iex
```

For more verbose output, run:

```powershell
$VerbosePreference = 'Continue'
irm pauw.io/bro | iex
```
